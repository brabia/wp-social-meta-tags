== Description ==
Social Meta Tags will generate social media meta data for all new posts that you publish. 
<br>This allows to share the right titles, descriptions, images and more when your posts are shared into Twitter, Facebook.

== Installation ==

- In your Admin, go to menu Plugins > Add
- Search for `Social Meta Tags`
- Click to install
- Activate the plugin
